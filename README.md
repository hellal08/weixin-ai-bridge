## 📦 What this app does

weixin-ai-bridge lets your WeChat account talk to AI tools through one simple setup. It acts like a bridge between WeChat and AI services such as Claude Code, OpenAI, Anthropic, and Ollama.

You can use it to:

- Send a message in WeChat and get an AI reply
- Connect one chat flow to different AI providers
- Keep your day-to-day chat use in one place
- Run it on your own Windows PC

## 🖥️ What you need

Before you start, make sure your Windows PC has:

- Windows 10 or Windows 11
- A stable internet connection
- A WeChat account that can sign in on desktop
- At least 4 GB of RAM
- Enough free disk space for the app and its data
- Access to the AI service you want to use

If you plan to use a local model, such as Ollama, your PC should have more memory and a bit more CPU power. If you use a cloud AI service, your normal home or office PC is enough.

## 🧰 Troubleshooting

### App will not open

- Try running it as administrator
- Check if Windows blocked the file
- Re-download the package if the file looks broken
- Make sure you extracted the ZIP file first

### WeChat does not reply

- Check that WeChat is signed in on desktop
- Make sure the bridge app is running
- Check the AI provider settings
- Test your API key again
- Restart both WeChat and the bridge

### AI replies are slow

- Cloud services can slow down when the network is busy
- Local models can slow down on older PCs
- Close heavy apps if your PC is under load
- Try a smaller model if you use Ollama

### Connection fails

- Confirm the Base URL is correct
- Check for firewall rules
- Check your proxy settings
- Make sure the service allows requests from your PC
